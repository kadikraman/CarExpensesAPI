Car expenses API

An interface to any and all car-related expenses from the car's initial purchase to the eventual goodbye.

This is a pet project that was born due to my interest in data visualisation and curiosity of 'how much does it actually cost to own a car'.

GET expense_types

Resource url: http://carexpensesapi.herokuapp.com/api/expense_types

Returns a collection containing all the registred expense types.

GET expenses

Resource url: http://carexpensesapi.herokuapp.com/api/expenses

Returns a collection of expenses (optionally filtered by the filter parameters).

type
Type of the expense (e.g. petrol).
date
Date of the expense (e.g. 2015-07-25).

GET expenses/:id

Resource url: http://carexpensesapi.herokuapp.com/api/expenses/:id

Returns the expense with the given id.
